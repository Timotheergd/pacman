# Projet : Pacman-like

Projet pour le Module électif Langage C - IMT Nord Europe

## Sujet

Concevoir un Pacman-like en C

## Comment lancer le jeu ?

### Sur Windows ou Mac

Aucune idée :(

### Sur Linux

- Ouvrir un terminal et se placer dans le dossier *./pacman*
- Taper la commande : ```$make all```
- S'amusez :)
